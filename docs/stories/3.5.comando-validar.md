## User Story
> **Como** operador, **quero** rodar `importacao-geo validar --mapeamento mapping.json`, **para** checar completude do JSON antes de tentar `importar`.

## Acceptance Criteria

| # | Critério |
|---|---|
| AC1 | Comando aceita `--mapeamento` (default `./mapping.json`) |
| AC2 | Invoca `MapeamentoValidador` e imprime relatório formatado em PT |

| AC4 | Se inválido: lista de pendências agrupadas por coluna + exit 1 |
| AC5 | **Não** toca o banco (comando puramente local) |
| AC6 | Teste unit do comando (sem Testcontainers) |
