## User Story
> **Como** sistema, **quero** classificar cabeçalhos de uma planilha em 3 categorias (código, fixa, dinâmica), **para** dirigir o auto-mapeamento.

## Acceptance Criteria

| # | Critério |
|---|---|
| AC1 | `ClassificadorColunas.classificar(List<String> headers, String nomeColunaCodigo, Set<String> colunasFixasConhecidas)` retorna `Classificacao(String codigo, Map<String,String> fixas, List<String> dinamicas)` |
| AC2 | `codigo` é o header que casa (exato, case-insensitive) com `nomeColunaCodigo` |
| AC3 | Cabeçalho em `colunasFixasConhecidas` vai para `fixas` (key = header da planilha, value = coluna da tabela) |
| AC4 | Cabeçalhos não classificados vão para `dinamicas` |
| AC5 | Se `nomeColunaCodigo` não for encontrada → `ImportacaoException` com mensagem PT |
| AC6 | Se o mesmo header aparecer duplicado → `ImportacaoException` |
| AC7 | Lista de colunas fixas é externa (config ou parâmetro) — **não hardcoded** no classificador |
| AC8 | Testes unit cobrindo happy / ambíguo / vazio / duplicado |
