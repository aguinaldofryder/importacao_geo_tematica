## User Story
> **Como** operador, **quero** validar a conexão ao banco antes de qualquer operação, **para** falhar-rápido se a configuração estiver errada.

## Acceptance Criteria

| # | Critério |
|---|---|
| AC1 | Datasource Quarkus Agroal configurado para PostgreSQL (chaves do AC4 da 1.4) |
| AC2 | `validar-conexao` injeta `AgroalDataSource`, executa `SELECT 1`, imprime `✓ Conexão OK (host:port, base=X, usuário=Y, versão PG=Z.W)` em sucesso |
| AC3 | Em falha, imprime `✗ Falha: <motivo amigável em PT>` e retorna exit `1` |
| AC4 | Timeout de conexão configurável (default 5 s) |
| AC5 | Mensagens em português; senha **nunca** aparece em stdout/stderr/log |
| AC6 | Teste integração com Testcontainers (PostgreSQL) cobrindo: sucesso, servidor inatingível, credencial inválida, timeout |
| AC7 | Exit codes: 0 sucesso, 1 qualquer falha (FR-16 básico) |
