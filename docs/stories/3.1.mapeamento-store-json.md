## User Story
> **Como** sistema, **quero** persistir e ler `mapping.json` em JSON tipado, **para que** a fase de mapeamento seja editável pelo usuário entre `mapear` e `importar`.

## Acceptance Criteria

| # | Critério |
|---|---|
| AC1 | Records do domínio de mapeamento criados em `mapeamento/`: `Mapeamento`, `ColunaDinamica` (com `status`, `idcampo`, `tipo`, `alternativas`, `motivo`, `sugestoes`) |
| AC2 | Enum `StatusMapeamento { MAPEADO, PENDENTE }` |
| AC3 | `MapeamentoStore.salvar(Mapeamento m, Path arquivo)` grava JSON UTF-8, pretty-printed |
| AC4 | `MapeamentoStore.carregar(Path arquivo)` retorna `Mapeamento` tipado; falha com mensagem PT se arquivo corrompido ou ausente |
| AC5 | Schema do JSON conforme Arquitetura §5 (exemplos `TIPO_MURO`, `COLUNA_NOVA`) |
| AC6 | `@RegisterForReflection` em todos os records |
| AC7 | Testes unit: round-trip (salvar e carregar), arquivo ausente, arquivo corrompido, JSON sem campo obrigatório |
