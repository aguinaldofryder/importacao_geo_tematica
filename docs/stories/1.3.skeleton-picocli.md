### Estrutura de um stub

```java
package br.com.arxcode.tematica.geo.cli;

import io.quarkus.picocli.runtime.annotations.TopCommand;
import picocli.CommandLine;

@CommandLine.Command(
    name = "mapear",
    description = "Gera o arquivo mapping.json com o mapeamento de colunas."
)
public class MapearCommand implements Runnable {
    @Override
    public void run() {
        System.out.println("[TODO Story 3.4] comando mapear ainda não implementado");
    }
}
```

### Padrão de teste (`@QuarkusMainTest`)

```java
@QuarkusMainTest
class MapearCommandTest {

    @Inject
    CommandLine.IFactory factory;

    @Test
    void deveRetornarExitZeroEImprimirTodo() {
        var cmd = new CommandLine(new MapearCommand(), factory);
        int exitCode = cmd.execute();
        assertThat(exitCode).isZero();
    }
}
```

> **Nota:** usar `quarkus-junit5` (já no BOM); não requer `@QuarkusTest` standalone para testes de CLI puro.
