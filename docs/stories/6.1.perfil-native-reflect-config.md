## User Story
> **Como** equipe, **quero** conseguir compilar o binário nativo em dev local (Linux), **para** validar o suporte a native-image **antes** de ligar o CI.

## Acceptance Criteria

| # | Critério |
|---|---|
| AC1 | `./mvnw package -Pnative` produz binário nativo funcional no Linux local |
| AC2 | Todos os records/POJOs usados por Jackson ou JDBC anotados `@RegisterForReflection` (ou registrados via classe marker) |
| AC3 | `src/main/resources/META-INF/native-image/` contém `reflect-config.json` e `resource-config.json` se necessário |
| AC4 | Binário roda `validar-conexao` contra Testcontainers |
| AC5 | Binário roda `mapear` e `importar` contra fixtures (smoke test manual) |
| AC6 | Binário final ≤ 100 MB (NFR-05); startup ≤ 500 ms (NFR-03) — medido e documentado |
| AC7 | `DEVELOPING.md` descreve como rodar `-agentlib:native-image-agent` para descobrir reflexões faltantes |
